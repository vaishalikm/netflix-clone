Netflix Clone :
A full-stack movie streaming web application inspired by Netflix, built using Flask, Python, MySQL/SQLite, HTML, CSS, and JavaScript.

Overview :
Netflix Clone is a web-based movie streaming platform that allows users to browse movies, watch videos, create personalized watchlists, and manage their accounts. The application includes secure user authentication and an admin dashboard for managing movies and users.

 Features:
   1.User Registration and Login
   2. Secure Password Authentication
   3. Browse Movies by Category
   4.Video Streaming Functionality
   5. Watchlist Management
   6. Admin Dashboard
   7.Responsive User Interface
   8. MySQL/SQLite Database Support

Installation :
1. Clone the repository :

bash
git clone https://github.com/vaishalikm/netflix-clone.git

2. Navigate to the project folder :
bash
cd netflix-clone

4. Install dependencies :
bash
pip install -r requirements.txt

4. Run the application :
bash
python app.py

5. Open the application in your browser :
text
http://localhost:5000
